🚀 HOW TO RUN (Copy & Paste)
═══════════════════════════════════════════════════════════════════════════

TERMINAL 1:
  cd frontend-mobile
  flutter pub get

TERMINAL 2:
  cd backend
  npm install
  npm run start:dev

TERMINAL 3:
  docker-compose up -d

TERMINAL 1 AGAIN:
  flutter run


👀 HOW TO VIEW THE APP INTERFACE
═══════════════════════════════════════════════════════════════════════════

Option 1: Emulator/Device (Best for development)
  └─ flutter run
     App launches on screen in real-time
     Press 'r' to hot reload code changes instantly
     Press 'R' to restart app (reset state)

Option 2: Web Browser
  └─ flutter run -d web
     Opens http://localhost:54098
     Use Chrome DevTools (F12) to inspect

Option 3: Advanced DevTools
  └─ devtools
     Access http://localhost:9100
     Inspect widget tree, network, performance

Option 4: Physical Phone
  └─ Connect via USB
     flutter run
     App runs on your actual phone

Option 5: Screenshots
  └─ flutter screenshot
     Saves PNG of current app state
