## Introduction

## 📍 SCENARIO 1: Quick Browse & Search

**Time: 3 minutes**
**User Type:** Anonymous Visitor

### Journey Map
```
Launch App
    ↓
Browse Screen (Sees listings grid)
    ↓
Search for "phone"
    ↓
View filtered results
    ↓
Filter by city "Baghdad"
    ↓
Tap to view iPhone listing
    ↓
See listing details
    ↓
Go back
```

### Steps to Execute

1. **Launch the app**
   - Shows Browse Screen with 20+ listings
   - Say: "Users see a clean grid of available listings"

2. **Search for "phone"**
   - Type in search box at top
   - Results auto-filter
   - Say: "Real-time search with instant results"

3. **Filter results further**
   - Click city dropdown: Select "Baghdad"
   - Results narrow down to Baghdad phones
   - Say: "Easy filtering by city, category, or custom criteria"

4. **View listing details**
   - Tap iPhone 13 Pro card
   - Show full details screen
   - Say: "Each listing has complete info, photos, and seller details"

5. **Try to save (Shows login requirement)**
   - Click heart icon to save
   - See login prompt
   - Say: "Users can save listings for later (requires login)"

6. **Go back**
   - Click back button
   - Return to Browse Screen

### Key Points to Mention
- ✅ Search works in real-time
- ✅ Multiple filter options available
- ✅ Responsive grid layout
- ✅ Smooth navigation
- ✅ Authentication gates certain features

## 📍 SCENARIO 6: Category & City Filtering

**Time: 3 minutes**
**User Type:** Specific searcher

### Journey Map
```
Browse with defaults
    ↓
Use category filter
    ↓
Narrow to specific category
    ↓
Add city filter
    ↓
See filtered results
    ↓
Try different combinations
```

### Steps to Execute

1. **Start on Browse Screen**
   - Show grid of all listings
   - Say: "20+ listings across all categories"

2. **Use category filter**
   - Click category dropdown (top)
   - Select "Cars"
   - Results update instantly
   - Show only car listings
   - Say: "Category filtering narrows results instantly"

3. **Add city filter**
   - Click city dropdown
   - Select "Baghdad"
   - Results narrow to Baghdad cars only
   - Say: "Location filtering for local buyers"

4. **Use search together**
   - Type "Toyota" in search
   - Results show only Toyota cars in Baghdad
   - Say: "Powerful combination of search + filters"

5. **Try different filter combinations**
   - Change category to "Real Estate"
   - Change city to "Erbil"
   - See Erbil real estate listings
   - Say: "Explore different categories and locations"

6. **Reset filters**
   - Clear search box
   - Select "All" categories
   - Select "All" cities
   - See full listing grid again
   - Say: "Easy reset to browse everything"

### Key Points to Mention
- ✅ Real-time filter updates
- ✅ Multiple filter options (Category, City, Search)
- ✅ Combined filtering capability
- ✅ Instant results refresh
- ✅ Easy to explore different options
